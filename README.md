
# 🌦️ climaTrackBot

**climaTrackBot** est un bot Discord écrit en Python permettant d'afficher la météo en temps réel pour n'importe quelle ville du monde, en se connectant à l'API OpenWeatherMap.  
Le projet est pensé pour être simple, évolutif, et facilement déployable.

---

## 🚀 Fonctions principales

- Commande `!meteo <ville>` : donne la météo en direct (température, ciel, humidité, vent)
- Lecture des variables sensibles depuis un fichier `.env`
- Projet modulaire et prêt à héberger sur un serveur
